## 13. Emergency Troubleshooting

### If the bureau page does not load

Check `index.html`, `CNAME`, repository Pages settings, and `sydney.cgnnews.net` DNS.

### If Weather or Sports routes do not load

Check the `weather/` and `sports/` directories where present, plus any static route links in `index.html`.

### If a bureau article route appears

Remove it unless there is a separately approved static page. This repository is not a dynamic bureau article publishing system.

### If a reporter attribution link is needed

Use the bureau root `https://sydney.cgnnews.net/` as the public attribution destination unless a specific approved reporter profile route exists.

### If search engines show stale metadata

Check `sitemap.xml`, `robots.txt`, page metadata, and the GitHub Pages deployment status.
